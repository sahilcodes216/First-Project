## Small Guess The Number code(P2)
# I created a "guess the number" small game using Python.
In this you have to guess the exact number, what that child think.
<br>The number is in between 1 to 100.
<br>If you guess a number lower than the exact number, then you will get "Higher the Number" as a hint.
<br>And, if you guess a number higher than the exact number, then you will get "Lower the number" as a hint.
<br>This loop will repeat untill you not guess the exact number.
